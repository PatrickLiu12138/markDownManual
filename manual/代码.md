# Markdown代码

如果是段落上的一个函数或片段的代码可以用反引号把它包起来（`），例如：

```
`printf()` 函数
```

效果：

> `printf()` 函数

## 代码区块
代码区块使用 4 个空格或者一个制表符（Tab 键），例如：
```
    printf()
    printf()
```

效果：

>     printf()
>     printf()

你也可以用 ``` 包裹一段代码，并指定一种语言（也可以不指定,指定语言后可显示代码高亮）：

    ```javascript
    $(document).ready(function () {
        alert('RUNOOB');
    });
    ```

效果：
> ```javascript
> $(document).ready(function () {
>     alert('RUNOOB');
> });
> ```
