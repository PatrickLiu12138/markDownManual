# Markdown 表格

Markdown 制作表格使用 | 来分隔不同的单元格，使用 - 来分隔表头和其他行。

语法格式如下：

```
|  表头   | 表头  |
|  ----  | ----  |
| 单元格  | 单元格 |
| 单元格  | 单元格 |
```

效果：

> |  表头   | 表头  |
> |  ----  | ----  |
> | 单元格  | 单元格 |
> | 单元格  | 单元格 |

## 对齐方式 

我们可以设置表格的对齐方式：

* `-:` 设置内容和标题栏居右对齐。
* `:-` 设置内容和标题栏居左对齐。
* `:-:` 设置内容和标题栏居中对齐。

示例：

```
| 左对齐 | 右对齐 | 居中对齐 |
| :-----| ----: | :----: |
| 111 | 21 | 321231 |
| 24241242442 | 3213123 | 4 |
```

效果：
> | 左对齐 | 右对齐 | 居中对齐 |
> | :-----| -----: | :----: |
> | 111 | 21 | 321231 |
> | 24241242442 | 3213123 | 4 |
